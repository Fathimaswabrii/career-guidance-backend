# 🎯 AI-Based Career Guidance System

An intelligent web application that helps students choose the right career path using **Machine Learning, Aptitude Analysis, and AI Chatbot support**.

---

## 🚀 Overview

Many students struggle to decide:
- Which stream to choose after 10th?
- Which course after 12th?
- Which career suits them best?

This system solves that problem using:
- Aptitude test (based on 8 intelligence types)
- Machine Learning prediction
- Personalized recommendations

---

## 🧠 Key Features

✅ Aptitude Test (Multiple Intelligence Based)  
✅ Career Prediction using Machine Learning  
✅ Stream Recommendation (Science / Commerce / Arts)  
✅ Course Suggestions  
✅ Skill Improvement Suggestions  
✅ AI Chatbot for career queries  
✅ User Dashboard with analytics  
✅ Secure JWT Authentication

## ⚙️ How It Works

1. User registers and logs in  
2. Creates profile (interests, skills)  
3. Takes aptitude test  
4. System analyzes 8 intelligence scores  
5. ML model predicts best career  
6. System recommends:
   - Career
   - Stream
   - Courses
   - Skills to improve  

---

## 🤖 Machine Learning

- Input: 8 intelligence scores  
- Model Used: Random Forest Classifier  
- Accuracy: ~98%  
- Approach:
  - Data cleaning & preprocessing  
  - Feature scaling  
  - Model training & evaluation  
  - Best model selected and deployed
