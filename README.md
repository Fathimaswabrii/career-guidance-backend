# AI-Based Career Guidance System

An intelligent web-based platform that helps students choose suitable career paths using Artificial Intelligence and Machine Learning.

The system analyzes student aptitude, interests, and skills to provide personalized career recommendations, stream suggestions, course guidance, and skill improvement recommendations.

---

# Project Overview

This project is designed to help students make better career decisions through a smart recommendation system.

The platform allows users to:

- Register and create profiles
- Take aptitude and intelligence-based tests
- Receive AI-powered career predictions
- Get stream and course recommendations
- Identify weak skill areas
- Interact with an AI career chatbot

The system combines:

- Machine Learning
- Rule-Based Recommendation Logic
- Full-Stack Web Development
- AI Chatbot Integration

# Machine Learning

The system uses a Random Forest Classifier trained on student intelligence data.

## Workflow

- Collect aptitude scores
- Combine interests using hybrid logic
- Normalize data using MinMaxScaler
- Predict careers using ML model
- Return top career matches with percentage
